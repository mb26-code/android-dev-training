# Android Dev Training

A collection of Android projects built during my Computer Science master's degree to practice mobile development fundamentals.
This repository is structured in a way such that each sub-directory is an Android Studio project (without certain files like generated build artifacts and user-specific configurations).

Note: I'll add previews or demos for each app to this repository in the future.

### How to run/test

1.  Clone this repository.
2.  Open Android Studio.
3.  Select **File > Open** (or **Open an existing project**).
4.  Navigate to and select one of the specific project directories to open and run it (do not open the root `android-dev-training` directory in Android Studio, as it is not a valid project).
